## Architecture
Somneria's assets are made in a mix of Blender and Krita and the Game itself made entirely in [Godot](https://godotengine.org/) and `GDScipt`, meaning it is entirely cross-platform.

## Development

Prerequisites:
* Godot
* Visual Studio Code with the `godot-tools` extension installed

1.
```bash
git clone https://github.com/Kaympe20/somneria
```
2. Import the `project.godot` file into Godot

![](https://files.catbox.moe/0dvt2t.png)

![](https://files.catbox.moe/ox92pj.png)

3. Edit Away

## Deployment
1. Download the desired templates by navigating to Editor > Manage Export Templates

![](https://docs.godotengine.org/en/stable/_images/export_templates.webp)

2. Navigate to Project > Export

![](https://files.catbox.moe/r8pknc.png)

3. Pick your desired preset and export as needed

![](https://files.catbox.moe/urrt9e.png)
